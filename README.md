📌 Overview

This is a simple static flower shop website built using HTML, CSS, and JavaScript.
The project includes multiple pages: Home, Gallery, Order, and About Us.

🛠️ Technologies Used

HTML5 – structure of the website

CSS3 – layout and visual styling

JavaScript – date display, scroll animation, form validation

jQuery + jquery-sakura plugin – falling petals animation

Google Fonts (Montserrat)

▶️ How to Open the Website

You can run it in two ways:

1️⃣ Open directly

Double-click index.html

The site opens in any browser
(Simple and works for static websites)

2️⃣ Using VS Code + Live Server

Recommended for video autoplay support.

Install the Live Server extension

Right-click index.html

Click “Open with Live Server”
